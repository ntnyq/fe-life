# 小技巧

# Require.resolve

通过使用 require.resolve API 可以判断是否存在此依赖。

若参数是内置的依赖，则返回依赖的名字。

console.log(require.resolve('http')) // => http

若参数是下载的依赖，则返回 package.json 里 main 字段文件的绝对路径。

console.log(require.resolve('vue')) // => Your_path/cwd_path/module_path/main_file_path

若参数表示的依赖不存在，则报错提示
