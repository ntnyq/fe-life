# vue-i18n

> 基于 vue-i18n v9。

## 非 SFC 中使用 i8n

在非 `.vue` 文件中使用 `useI18n` 会被警告，只能在 `setup` 方法内使用。

```ts
// src/i18n/index.ts
import { createI18n } from 'vue-i18n'

const messages = {
  zh: {
    hello: `你好`,
  },
}

const i18n = createI18n({
  legacy: false,
  locale: `zh`,
  messages,
})

export default i18n
```

```ts
// tip.ts
import i18n from '@/i18n'

const { t } = i18n.global

export function showHello () {
  console.log(t(`hello`))
}
```
