# Babel 编译中文转 Unicode

在 js 中编写了模板字符串，经过 Babel 转译后，发现将中文字符转为了 Unicode 字符，影响阅读。

解决方案，在 Babel 的配置中添加如下配置：

{
  "generatorOpts": {
    "jsescOption": {
      "minimal": true
    }
  }
}
