# Gulp

Gulp 是一款基于流的自动化构建工具。与 Webpack 不同，它是一款Task Runner。

# 踩坑记录

# gulp-sass 报错

[17:49:39] Starting 'scss'...
[17:49:39] Finished 'scss' after 3.85 ms
Error in plugin "sass"
Message:
    src\scss\style.scss
Error: File to import not found or unreadable: ./_utils/_style.
        on line 3 of src/scss/style.scss
>> @import "./_utils/_style";

解决：

  1. 在 sublime 的配置项中增加"atomic_save": true。
  2. 使用gulp-wait。

# Gulp 插件

以gulp-*为规则命名的插件。

# 样式处理

  • gulp-sass (opens new window) SASS && SCSS 预编译
  • gulp-less (opens new window) Less 预编译
  • gulp-postcss (opens new window) Postcss 后处理
  • gulp-autoprefixer (opens new window) 浏览器私有前缀、兼容样式补全等

# ES6 编译

  • gulp-babel (opens new window) ES6 及以后版本编译为 ES5

    需要搭配babel-core与babel-preset-env使用

# 模板引擎

  • gulp-pug (opens new window) Pug，原名 Jade 的 Gulp 插件，使用模板引擎编写静态页面
  • gulp-ejs (opens new window) Ejs 模板引擎的 Gulp 插件
  • gulp-data (opens new window) 通过读取 json 文件为模板引擎注入数据

# 代码美化

  • gulp-html-beautify (opens new window) HTML 代码格式美化，代码强迫症的选择
  • gulp-prettier (opens new window) Prettier 工具的 Gulp 插件，JS 等代码美化，可使用插件支持别的语言

# 性能优化

  • gulp-base64 (opens new window) 将样式表中的背景图转化为 base64 格式的 dataURL
  • gulp-cssnano (opens new window) CSS 文件压缩，移除注释等
  • gulp-uglify (opens new window) JS 文件压缩，UglifyJS 的 Gulp 插件

# 文档目录

  • gulp-rename (opens new window) 文件名修改，如增加前缀、后缀、修改路径等
  • gulp-size (opens new window) 输出文件的大小
  • gulp-clean (opens new window) 迭代清空指定目录及其下所有文件
  • gulp-concat (opens new window) 多文件合并
  • gulp-sourcemaps (opens new window) SourceMap 文件生成

# 其他插件

  • gulp-load-plugins (opens new window) 自动挂载所有 Gulp 插件，强烈推荐使用
  • gulp-sequence (opens new window) 同步顺序执行 Gulp Task
  • gulp-wait (opens new window) 在 Gulp 的管道流中加入延迟事件，阻塞任务
  • gulp-header (opens new window) 文件元信息添加，如版本、作者、说明文档、开源协议等
  • gulp-if (opens new window) 流程控制

# NPM 插件

  • browser-sync (opens new window) 基于 WebSocket 的浏览器自动刷新
  • mkdirp (opens new window) 迭代创建目录
